  • HTTP иҜ·жұӮеҸ‘йҖҒ & е“Қеә”жҺҘж”¶

  • еёғеұҖ Layout вҶ’ з»ҳеҲ¶ Paint вҶ’ еҗҲжҲҗ Composite
  • JS жү§иЎҢгҖҒдәӢд»¶з»‘е®ҡгҖҒйЎөйқўдәӨдә’е°ұз»Ә

жөҸи§ҲеҷЁиҫ“е…Ҙ URLпјҢе…ҲеҒҡжң¬ең°йў„еӨ„зҗҶ вҖӢ

URL иҜӯжі•и§Јжһҗ вҖӢ

жөҸи§ҲеҷЁе…ҲжӢҶи§Ј URL еҗ„йғЁеҲҶпјҡ

еҚҸи®® http/httpsгҖҒеҹҹеҗҚгҖҒз«ҜеҸЈгҖҒи·Ҝеҫ„гҖҒеҸӮж•°гҖҒе“ҲеёҢ #

жөҸи§ҲеҷЁзј“еӯҳдјҳе…Ҳзә§еҲӨж–ӯ вҖӢ

йЎәеәҸпјҡ

Memory Cache вҶ’ Disk Cache вҶ’ ејәеҲ¶зј“еӯҳ вҶ’ еҚҸе•Ҷзј“еӯҳ

  • е…ҲзңӢеҶ…еӯҳзј“еӯҳпјҡеҪ“еүҚ tab еҶ…еӯҳйҮҢжңүжІЎжңүпјҢжңүзӣҙжҺҘжӢҝ
  • еҶҚзңӢзЎ¬зӣҳзј“еӯҳпјҡжң¬ең°жңүжІЎжңүиҝҮжңҹзј“еӯҳ
  • жІЎзј“еӯҳжүҚиө°зҪ‘з»ңиҜ·жұӮ

еүҚз«Ҝе…іиҒ”пјҡ йқҷжҖҒиө„жәҗзј“еӯҳзӯ–з•ҘгҖҒCache-ControlгҖҒETagгҖҒLast-Modified йғҪжҳҜеңЁиҝҷдёҖжӯҘз”ҹж•ҲгҖӮ

HSTS / йў„и§Јжһҗ вҖӢ

жөҸи§ҲеҷЁиҮӘеёҰ HSTS еҲ—иЎЁпјҢејәеҲ¶и·іиҪ¬ HTTPSпјӣ

еҗҢж—¶и§ҰеҸ‘ DNS йў„и§Јжһҗ <link rel="dns-prefetch">гҖӮ

DNS еҹҹеҗҚи§ЈжһҗпјҲеҹҹеҗҚ вҶ’ IPпјү вҖӢ

жөҸи§ҲеҷЁдёҚзҹҘйҒ“еҹҹеҗҚеңЁе“ӘпјҢеҝ…йЎ»зҝ»иҜ‘жҲҗжңҚеҠЎеҷЁ IPгҖӮ

йҖ’еҪ’жҹҘиҜўжөҒзЁӢ

жөҸи§ҲеҷЁзј“еӯҳ вҶ’ зі»з»ҹзј“еӯҳ вҶ’ и·Ҝз”ұеҷЁзј“еӯҳ вҶ’ иҝҗиҗҘе•Ҷ DNS вҶ’ ж №еҹҹеҗҚжңҚеҠЎеҷЁ вҶ’ йЎ¶зә§еҹҹеҗҚжңҚеҠЎеҷЁ вҶ’ зӣ®ж ҮеҹҹеҗҚжңҚеҠЎеҷЁ

еүҚз«Ҝе…іиҒ”зӮ№

  • йЎөйқўеӨҡеҹҹеҗҚиҜ·жұӮеӨҡпјҢDNS иҖ—ж—¶й«ҳ
  • дјҳеҢ–пјҡ<link rel="dns-prefetch" href="https://xxx.com"> йў„и§Јжһҗ
  • еҹҹеҗҚж”¶ж•ӣпјҡеҮҸе°‘йЎөйқўи·ЁеҹҹеҗҚж•°йҮҸпјҢеҮҸе°‘ DNS ејҖй”Җ

TCP дёүж¬ЎжҸЎжүӢ + HTTPS TLS жҸЎжүӢ вҖӢ

TCP дёүж¬ЎжҸЎжүӢ вҖӢ

е»әз«ӢеҸҜйқ иҝһжҺҘпјҢдҝқиҜҒж•°жҚ®дёҚдёўгҖҒдёҚд№ұеәҸгҖӮ

HTTPS TLS еҚҸе•ҶпјҲ4 ж¬ЎжҸЎжүӢпјү вҖӢ

  • иҜҒд№Ұж ЎйӘҢпјҲеҹҹеҗҚеҢ№й…ҚгҖҒжҳҜеҗҰиҝҮжңҹгҖҒжҳҜеҗҰеҸҜдҝЎпјү
  • еҚҸе•ҶеҠ еҜҶз®—жі•гҖҒз”ҹжҲҗдјҡиҜқеҜҶй’Ҙ
  • д№ӢеҗҺжүҖжңүж•°жҚ®еҠ еҜҶдј иҫ“

жңүдәӣзҪ‘з«ҷйҰ–ж¬Ўжү“ејҖж…ўпјҢе°ұжҳҜ TLS иҜҒд№Ұж ЎйӘҢгҖҒжҸЎжүӢиҖ—ж—¶пјӣ дјҳеҢ–пјҡHTTPS дјҡиҜқеӨҚз”ЁгҖҒHTTP/2гҖҒHTTP/3гҖӮ

еҸ‘иө· HTTP иҜ·жұӮ & жҺҘж”¶е“Қеә” вҖӢ

жһ„е»әиҜ·жұӮжҠҘж–Ү вҖӢ

иҜ·жұӮиЎҢгҖҒиҜ·жұӮеӨҙгҖҒиҜ·жұӮдҪ“гҖӮ

  • иҜ·жұӮеӨҙ User-AgentгҖҒCookieгҖҒRefererгҖҒOrigin
  • и·Ёеҹҹе°ұжҳҜжөҸи§ҲеҷЁеңЁиҝҷдёҖжӯҘеҒҡеҗҢжәҗзӯ–з•ҘжӢҰжҲӘ

жңҚеҠЎеҷЁеӨ„зҗҶ & иҝ”еӣһе“Қеә” вҖӢ

е“Қеә”иЎҢгҖҒе“Қеә”еӨҙгҖҒе“Қеә”дҪ“пјҲHTML жәҗз Ғпјү

еүҚз«Ҝе…ій”®зҹҘиҜҶзӮ№ вҖӢ

  • и·Ёеҹҹ CORSгҖҒйў„жЈҖ OPTIONS иҜ·жұӮ
  • зҠ¶жҖҒз Ғ 200/301/302/304/404/502
  • е“Қеә”еӨҙжҺ§еҲ¶зј“еӯҳгҖҒи·ЁеҹҹгҖҒеҶ…е®№зұ»еһӢ

жөҸи§ҲеҷЁи§Јжһҗ HTMLпјҢжһ„е»ә DOM ж ‘ вҖӢ

еӯ—иҠӮжөҒ вҶ’ иҜҚжі•и§Јжһҗ вҶ’ иҜӯжі•и§Јжһҗ вҶ’ DOM ж ‘

жөҸи§ҲеҷЁдёҖиҫ№дёӢиҪҪ HTMLгҖҒдёҖиҫ№жөҒејҸи§ЈжһҗпјҢдёҚз”Ёзӯүе…ЁйғЁдёӢиҪҪе®ҢгҖӮ

е…ій”®и§„еҲҷ

  • йҒҮеҲ° <link> еӨ–й“ҫ CSSпјҡе№¶иЎҢдёӢиҪҪпјҢдёҚйҳ»еЎһ HTML и§ЈжһҗпјҢдҪҶйҳ»еЎһжёІжҹ“
  • йҒҮеҲ° <script> еӨ–й“ҫ JSпјҡйҳ»еЎһ HTML и§ЈжһҗгҖҒйҳ»еЎһжёІжҹ“
    • жІЎжңү defer/async ж—¶пјҢйЎөйқўдјҡеҚЎеңЁиҝҷ
  • deferпјҡеҗҺеҸ°дёӢиҪҪпјҢйЎәеәҸжү§иЎҢпјҢзӯү HTML и§Јжһҗе®ҢеҶҚи·‘
  • asyncпјҡеҗҺеҸ°дёӢиҪҪпјҢдёӢиҪҪе®Ңз«ӢеҲ»д№ұеәҸжү§иЎҢ

JS ж”ҫеә•йғЁгҖҒdefer/asyncгҖҒеҠЁжҖҒ import жҢүйңҖеҠ иҪҪпјҢйғҪжҳҜдёәдәҶдёҚйҳ»еЎһ HTML и§ЈжһҗгҖӮ

и§Јжһҗ CSSпјҢз”ҹжҲҗ CSSOM ж ‘ + жёІжҹ“ж ‘ вҖӢ

  • жөҸи§ҲеҷЁи§Јжһҗ CSS вҶ’ з”ҹжҲҗ CSSOM ж ·ејҸи§„еҲҷж ‘
  • DOM + CSSOM еҗҲе№¶ вҶ’ жёІжҹ“ж ‘ Render Tree
    • еҸӘеҢ…еҗ«йңҖиҰҒжҳҫзӨәзҡ„иҠӮзӮ№
    • display: none дёҚдјҡиҝӣе…ҘжёІжҹ“ж ‘
    • visibility: hidden дјҡиҝӣжёІжҹ“ж ‘пјҢеҚ з©әй—ҙ

йЎөйқўжёІжҹ“дёүеӨ§жөҒзЁӢ вҖӢ

Layout еёғеұҖпјҲйҮҚжҺ’пјү вҖӢ

ж №жҚ®жёІжҹ“ж ‘пјҢи®Ўз®—жҜҸдёӘе…ғзҙ зІҫзЎ®е®Ҫй«ҳгҖҒдҪҚзҪ®гҖҒеҮ дҪ•еқҗж ҮгҖӮ

и§ҰеҸ‘йҮҚжҺ’еңәжҷҜпјҡ

  • ж”№еҸҳе®Ҫй«ҳгҖҒиҫ№и·қгҖҒе®ҡдҪҚгҖҒеӯ—дҪ“еӨ§е°Ҹ
  • иҺ·еҸ– offsetTop /offsetLeft/clientWidth зӯүеёғеұҖеұһжҖ§

еҫӘзҺҜйў‘з№ҒиҜ»еҶҷеёғеұҖеұһжҖ§пјҢеҜјиҮҙејәеҲ¶еҗҢжӯҘеёғеұҖгҖҒеӨ§йҮҸйҮҚжҺ’еҚЎйЎҝгҖӮ

Paint з»ҳеҲ¶пјҲйҮҚз»ҳпјү вҖӢ

жҢүз…§еёғеұҖз»“жһңпјҢеЎ«е……еғҸзҙ пјҡ

йўңиүІгҖҒиғҢжҷҜгҖҒйҳҙеҪұгҖҒж–Үеӯ—гҖҒеӣҫзүҮгҖӮ

еҸӘж”№йўңиүІгҖҒиғҢжҷҜ вҶ’ еҸӘйҮҚз»ҳпјҢдёҚйҮҚжҺ’пјҢжҖ§иғҪжӣҙеҘҪгҖӮ

Composite еӣҫеұӮеҗҲжҲҗ вҖӢ

жөҸи§ҲеҷЁжҠҠйЎөйқўеҲҶжҲҗеӨҡдёӘеҗҲжҲҗеӣҫеұӮпјҢдәӨз»ҷ GPU еҗҲжҲҗгҖӮ

ејҖеҗҜзӢ¬з«ӢеӣҫеұӮжқЎд»¶

  • transform: translateZ(0)
  • will-change
  • 3D transformгҖҒvideoгҖҒcanvas

иғҪз”Ё transform/opacity еҒҡеҠЁз”»пјҢз»қдёҚж”№ top/left еӣ дёә transform еҸӘиө° еҗҲжҲҗеұӮпјҢдёҚи§ҰеҸ‘йҮҚжҺ’гҖҒйҮҚз»ҳпјҢGPU еҠ йҖҹдёқж»‘гҖӮ

JS жү§иЎҢгҖҒDOMContentLoadedгҖҒLoad дәӢд»¶ вҖӢ

JS жү§иЎҢ вҖӢ

JS еҸҜд»Ҙж“ҚдҪң DOM/CSSпјҢдјҡйҮҚж–°и§ҰеҸ‘жёІжҹ“ж ‘гҖҒйҮҚжҺ’йҮҚз»ҳгҖӮ

DOMContentLoaded вҖӢ

HTML и§Јжһҗе®ҢгҖҒDOM жһ„е»әе®ҢжҲҗпјҢдёҚз”ЁзӯүеӣҫзүҮгҖҒж ·ејҸгҖҒи§Ҷйў‘

вҶ’ еүҚз«ҜйҖӮеҗҲеңЁиҝҷйҮҢеҲқе§ӢеҢ–дёҡеҠЎйҖ»иҫ‘гҖҒжҢӮиҪҪ Vue/React ж №е®һдҫӢ

window.onload вҖӢ

жүҖжңүиө„жәҗпјҲеӣҫзүҮгҖҒеӯ—дҪ“гҖҒи§Ҷйў‘гҖҒж ·ејҸпјүе…ЁйғЁеҠ иҪҪе®ҢжүҚи§ҰеҸ‘

вҶ’ йҖӮеҗҲеҒҡйЎөйқўз»ҹи®ЎгҖҒеҠ иҪҪе®ҢжҲҗеј№зӘ—

ж•ҙжқЎй“ҫи·ҜеҜ№еә” еүҚз«Ҝе·ҘзЁӢеёҲ зҡ„е·ҘдҪң вҖӢ

йҳ¶ж®өеүҚз«Ҝж—Ҙеёёе·ҘдҪң & дјҳеҢ–зӮ№
зј“еӯҳй…ҚзҪ® Cache-ControlгҖҒйқҷжҖҒиө„жәҗзј“еӯҳгҖҒе“ҲеёҢе‘ҪеҗҚ

HTML и§ЈжһҗJS ж”ҫеә•йғЁгҖҒdefer/asyncгҖҒйҒҝе…Қйҳ»еЎһ
